# Calvin Frontend

Vue 3 frontend for Calvin Dashboard.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Development

```bash
# Run tests
npm run test

# Lint
npm run lint

# Format
npm run format

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/    # Vue components
├── views/         # Page views
├── stores/        # Pinia stores
├── services/      # API services
├── composables/   # Vue composables
└── styles/        # CSS styles
```
